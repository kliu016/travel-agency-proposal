Questions We Need Answered

To ensure we build exactly what you need, we'll need to discuss:

1. Business Process Questions
  • What happens when a booking is created? Who gets notified?
  • How are invoices currently generated and sent?
  • What are the different user roles and what can each role do?
2. Data Relationship Questions
  • Can a Traveler exist without being part of a Group?
  • What happens to bookings if a Group is removed?
  • How do you handle booking modifications and cancellations?
3. Integration Questions
  • Do you need to integrate with any airline or hotel systems?
  • Are there any accounting systems we need to connect to?
  • Do you need automated email notifications?
4. Priority Questions
  • Which booking type should we implement first? (Air, Hotel, Car, etc.)
  • Are there any features in Airtable you don't use anymore?
  • What new features would help your business most?
